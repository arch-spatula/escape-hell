## 구현한 기능

1. 메인 페이지
   - 입장하기 버튼 누르면 레벨 버튼으로 스크롤
   - 레벨 버튼 클릭 시 해당 튜토리얼 페이지로 이동
   - 스택 이미지와 제목 불러오기
2. 튜토리얼 페이지
   - 헤더에 네비게이션과 드롭다운 적용
   - 드롭 다운에 네비게이션 적용
   - 레벨 제목, 스택 이미지, 스택 이름, 플레이리스트 주소 불러오기
   - 댓글, 동영상 로딩 중 스켈레톤 UI 보여주기
   - 댓글 CRUD, 수정, 삭제 시 비밀번호 입력 (supabase 사용)
   - 검증 버튼 클릭 시 체크 리스트 모달 띄우기
   - 체크 리스트에 체크 박스 토글 가능
